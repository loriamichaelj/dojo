# Quantitative Analysis

Turning data into testable claims about markets.

**In scope:** descriptive & inferential statistics, hypothesis testing, time-series
(ARIMA/GARCH, stationarity, cointegration), factor models, signal construction and
evaluation, and backtesting methodology (look-ahead bias, overfitting, multiple-testing
corrections, walk-forward validation).

**Pairs with:** [`../math/`](../math/) for theory and
[`../strategy-development/`](../strategy-development/) for productionizing signals.

> Each empirical note should state its data source, sample period, and how the result was
> validated against overfitting.
